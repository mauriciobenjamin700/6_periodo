Para instalar todas as dependências listadas em um arquivo `requirements.txt` de uma única vez, você pode usar o comando `pip` no terminal ou prompt de comando. Certifique-se de estar no diretório onde o arquivo `requirements.txt` está localizado. Aqui estão os passos:

1. Abra um terminal ou prompt de comando.

2. Navegue até o diretório onde o arquivo `requirements.txt` está localizado usando o comando `cd` (change directory). Por exemplo:
   
   ```
   cd /caminho/do/diretorio/com/o/requirements.txt
   ```

3. Execute o comando `pip` com o argumento `-r` seguido do nome do arquivo `requirements.txt`. Por exemplo:

   ```
   pip install -r requirements.txt
   ```

Isso fará com que o `pip` leia o arquivo `requirements.txt` e instale todas as dependências listadas nele de uma só vez. Certifique-se de que você tenha o `pip` instalado e configurado corretamente em seu ambiente Python. Se você estiver usando um ambiente virtual (recomendado), ative-o antes de executar o comando acima para garantir que as dependências sejam instaladas no ambiente correto.

Lembre-se de que é uma prática recomendada usar ambientes virtuais para isolar as dependências de diferentes projetos, a fim de evitar conflitos entre elas. Para criar um ambiente virtual, você pode usar o `virtualenv` ou o `venv`, dependendo da versão do Python que está usando.
